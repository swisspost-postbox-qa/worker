Swiss Post – Postbox QA (Prototype)

This is an early QA prototype to support OpenStreetMap mapping of amenity=post_box in Switzerland.

No automated edits.
Swiss Post data is used as an input for review only.

Method

  • Swiss Post public Standortsuche (Find + POI API)
  • OSM data via Overpass (amenity=post_box)
  • Spatial matching within a fixed radius
  • Human review required for all changes
