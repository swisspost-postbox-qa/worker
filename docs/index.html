Swiss Post – Postbox QA (Prototype)

This is an early QA prototype to support the maintenance of amenity=post_box data in OpenStreetMap (Switzerland).

No automated edits are performed.
Data from Swiss Post’s public location search is used for review and comparison only. OSM remains the authoritative record of on-the-ground reality.

Method (summary)

  • Swiss Post public location search (Find + POI endpoints)
  • OSM data via Overpass API (amenity=post_box)
  • Spatial comparison using a fixed matching radius
  • All results require human review before any OSM change

Project repository: github.com/swisspost-postbox-qa/worker
This project is community-driven and not affiliated with Swiss Post or the OpenStreetMap Foundation.

Legal considerations

Swiss Post data is used as a reference signal only. This project does not reproduce, redistribute or publish Swiss Post location data or content.

Only derived QA hints are generated (e.g. “please verify on the ground”). No Swiss Post datasets, tables or collection times are transferred into OpenStreetMap.

Any OSM edits resulting from this project are intended to be performed manually by human mappers, based on on-the-ground verification and local knowledge.
